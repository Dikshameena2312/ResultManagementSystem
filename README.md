## Set up your local environment

fork [this](https://github.com/YashJain2409/ResultManagementSystem) repository

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
   node app.js
```
